MiB Desk
=========


A console based to alien registration

For compiling [sbt](http://www.scala-sbt.org/) tool is required

Run the app with :-

```bash 
sbt run
```

To create the package run :-

```bash
sbt package
```

The jar can be downloaded for the release which can be executed using :-

```bash
scala mib_desk.jar
```
